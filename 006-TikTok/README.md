# App自动化

## 效果

抖音和tiktok能自动刷App评论。

## 实现
1. 使用android手机，能与电脑正常使用adb连接
2. 使用[uiautomator2](https://github.com/openatx/uiautomator2) 开源库。    
3. 借助：weditor 来获取元素（xpath等）。
